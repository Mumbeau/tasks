# Task Manager TUI

A lightweight Python application for managing tasks running on the curses library.

## Features
- Active task limiting (max 3 concurrent)
- Resize handling
- Scrolling with proportional scrollbar
- Arrow key navigation
- Mouse input
- Project grouping (in progress)

## Usage
Run the script using:
`python tasks.py`

## Requirements
- Python 3.10+
- 256-color terminal support
- UTF-8 Support
- **Windows Users:** Requires `pip install windows-curses`
